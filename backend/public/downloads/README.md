# Staff / Admin Downloads

This folder contains the **Admin Dashboard APKs** (`EMI-Admin-Dashboard`).

- **For Staff Only**: These apps provide control over customer devices.
- **Do Not Share**: Do not provide these links to customers.
- **Updates**: The `version.json` in this folder controls updates for the Admin app.

The Customer/User APKs are located in the parent `downloads` folder to facilitate QR code provisioning.
